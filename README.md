# dotfiles

[![Build Status](https://travis-ci.com/valbeat/dotfiles.svg?branch=master)](https://travis-ci.com/valbeat/dotfiles)

## Installing

```shell
$ git clone git@github.com:valbeat/dotfiles.git $HOME/dotfiles
$ cd $HOME/dotfiles
$ make install
$ zsh
```
